📌 Secure Password Manager (Python)

🔐 Overview

This project is a Secure Password Manager built using Python and cryptography libraries.
It encrypts user passwords before storing them, ensuring sensitive data is protected from unauthorized access.

The project was developed on Kali Linux to practice cybersecurity and secure coding techniques.

⸻

🎯 Features
	•	Encrypted password storage using Fernet cryptography
	•	Secure password retrieval
	•	Hidden password input (no typing visible)
	•	Automatic encryption key generation
	•	Command-line interface (CLI)

⸻

🛠️ Technologies Used
	•	Python 3
	•	Cryptography (Fernet)
	•	Linux / Kali Linux
	•	Virtual Environment (venv)

🚀 Usage
When you run the program, you will see:
1. Save password
2. Get password
3. Exit
•	Choose 1 to save a password
	•	Choose 2 to retrieve a password
	•	Choose 3 to exit
